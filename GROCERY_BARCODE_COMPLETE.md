## 🎯 How It Works Now

### Creating a Stock Lot:

**Request:**
```bash
POST /api/grocery/stock-lots
{
  "product_id": "...",
  "supplier_id": "...",
  "qty_in": 100,
  "cp": 50.00,
  "sp": 75.00,
  "batch_no": "BATCH-001",
  "expiry_date": "2025-12-31"
}
```

**Process:**
1. Generate UUID for lot_id
2. Generate barcode: `"LOT-L5K2M3N-AB4C"`
3. Render Code128 barcode image (PNG)
4. Upload to S3: `barcodes/{lot_id}.png`
5. Store in database with barcode fields

**Response:**
```json
{
  "success": true,
  "data": {
    "product": { ... },
    "supplier": { ... },
    "lot": {
      "lot_id": "a1b2c3d4-...",
      "barcode": "LOT-L5K2M3N-AB4C",
      "barcode_image_url": "barcodes/a1b2c3d4-....png",
      "qty_in": 100.0,
      "qty_remaining": 100.0,
      "cp": 50.00,
      "sp": 75.00,
      "batch_no": "BATCH-001",
      "expiry_date": "2025-12-31T00:00:00.000Z",
      "created_at": "2026-05-19T08:33:00.000Z",
      "product": { ... },
      "supplier": { ... }
    }
  }
}
```

## 📊 Barcode Details

### Barcode Value
- **Format**: `LOT-{timestamp_base36}-{random}`
- **Example**: `LOT-L5K2M3N-AB4C`
- **Unique**: Yes (enforced by database UNIQUE constraint)
- **Generated by**: `generateBarcodeValue()` in `utils/barcode.js`

### Barcode Image
- **Type**: Code128 barcode
- **Format**: PNG image
- **Library**: `bwip-js`
- **Scale**: 3x
- **Height**: 10 units
- **Text**: Included below barcode
- **Storage**: S3 (MinIO)
- **Path**: `barcodes/{lot_id}.png`
- **Generated by**: `renderBarcodeImage()` in `utils/barcode.js`

## 🌐 S3 Storage

### Configuration
- **Endpoint**: `s3-np1.datahub.com.np`
- **Bucket**: From `process.env.AWS_S3_BUCKET`
- **Path**: `barcodes/{lot_id}.png`
- **Content-Type**: `image/png`
- **Access**: Public read

### Accessing Barcode Image
```javascript
import { getS3Url } from '../utils/s3.js';

const imageUrl = getS3Url(lot.barcode_image_url);
// Returns: https://s3-np1.datahub.com.np/{BUCKET}/barcodes/{lot_id}.png
```

## 📱 Frontend Integration

### Display Barcode Image
```jsx
import { getS3Url } from './utils/s3';

function StockLotCard({ lot }) {
  const barcodeUrl = getS3Url(lot.barcode_image_url);
  
  return (
    <div>
      <h3>{lot.product.product_name}</h3>
      <p>Barcode: {lot.barcode}</p>
      <img 
        src={barcodeUrl} 
        alt={`Barcode ${lot.barcode}`}
        style={{ width: '200px' }}
      />
    </div>
  );
}
```

### Print Barcode Label
```jsx
function PrintBarcodeLabel({ lot }) {
  const barcodeUrl = getS3Url(lot.barcode_image_url);
  
  const handlePrint = () => {
    const printWindow = window.open('', '', 'width=400,height=300');
    printWindow.document.write(`
      <html>
        <head><title>Barcode Label</title></head>
        <body style="text-align: center; padding: 20px;">
          <h3>${lot.product.product_name}</h3>
          <p>Batch: ${lot.batch_no || 'N/A'}</p>
          <img src="${barcodeUrl}" style="width: 250px;" />
          <p>${lot.barcode}</p>
        </body>
      </html>
    `);
    printWindow.document.close();
    printWindow.print();
  };
  
  return <button onClick={handlePrint}>Print Label</button>;
}
```
